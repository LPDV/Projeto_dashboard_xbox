# Projeto_dashboard_xbox
Projeto para desenvolvimento de um dashboard utilizando o Excel. 

# Descrição do Projeto
Este projeto consiste na criação de um dashboard utilizando o software Excel com base em dados fornecida.  
O objetivo é analisar e visualizar dados de forma clara, facilitando a interpretação dos dados e possibilitando a tomada de decisões. 

# Dados Utilizados
Os dados utilizados no projeto foram fornecidos como parte do desafio e tratados diretamente no Excel, utilizando tabelas dinâmicas, gráficos, segmentações de dados e indicadores.

# Ferramentas Utilizadas
- Microsoft Excel
- GitHub para entrega do projeto

# Arquivo do Projeto
- `Dashboard_xbox.xlsx`

# Resultado Final
O dashboard apresenta uma visualização clara dos dados e resultados obtidos por meio das tabelas dinâmicas,  permitindo análises rápidas e eficientes.
